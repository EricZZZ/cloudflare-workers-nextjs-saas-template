# 如何使用博客系统

欢迎使用我们内置的博客系统！本指南将引导您了解开始使用我们强大而灵活的博客平台所需的一切知识。

## 开始使用

博客系统设计得直观易用。要访问博客，只需导航到您网站上的 `/blog` 路径。在那里，您将看到所有已发布的文章列表。

## 创建博客文章

创建博客文章非常简单：

1. 导航到项目中的 `content/blog` 目录
2. 创建一个带有语言代码的新文件夹（例如，`zh` 表示中文）
3. 创建一个包含您内容的 MDX 文件
4. 在文件顶部添加 frontmatter 元数据

以下是一个示例结构：

```mdx
---
title: "您的博客文章标题"
description: "您文章的简短描述"
createdAt: 2024-02-01T10:00:00Z
updatedAt: 2024-02-01T10:00:00Z
tags: ["标签1", "标签2"]
author: "您的姓名"
---

# 您的内容在这里

使用 Markdown 语法撰写您的博客文章...
```

## 博客功能

### 目录

我们的博客会为包含多个标题的文章自动生成目录。这使得读者可以轻松浏览较长的文章。

### 返回顶部按钮

对于较长的文章，我们提供了一个便捷的"返回顶部"按钮，当您向下滚动页面时会出现，让您能够快速回到顶部。

### 响应式设计

博客系统完全响应式，在移动设备、平板电脑和桌面上都能很好地工作。

## 多语言支持

我们的博客系统开箱即支持多种语言：

- 英语 (`en`)
- 中文 (`zh`)
- 日语 (`ja`)

要在不同语言中创建内容，只需将您的 MDX 文件放在 `content/blog` 下的相应语言文件夹中即可。

## 最佳实践

1. 使用描述性标题和元描述
2. 包含相关标签以更好地组织内容
3. 添加图片使您的文章更具吸引力
4. 正确使用标题层次结构（H1、H2、H3等）
5. 保持段落简短易读

## 结论

我们的博客系统设计得既强大又易于使用。凭借自动生成目录、多语言支持和响应式设计等功能，它为您提供了一切创建和维护专业博客所需的功能。
